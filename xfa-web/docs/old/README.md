# XFinder Archery Shop (documento antigo)

Este arquivo existe apenas por histórico (pasta `docs/old`). As instruções abaixo são para editar o frontend localmente, sem dependência de ferramentas externas.

## Como editar o frontend localmente

Pré-requisitos:

- Node.js + npm (ou pnpm/yarn, se você preferir)

Passos:

```sh
# 1) Clone o repositório
git clone <URL_DO_SEU_REPO>

# 2) Entre no frontend
cd xfa-web

# 3) Instale dependências
npm i

# 4) Rode em modo desenvolvimento
npm run dev
```

## Tecnologias principais

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
